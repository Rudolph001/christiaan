# AI Trading Assistant - System Requirements

## Software Requirements

### Required Software

1. **Node.js (Required)**
   - Version: 18.0.0 or higher (LTS recommended)
   - Download: https://nodejs.org/
   - Purpose: JavaScript runtime for the application
   - Installation: Choose "Add to PATH" during setup

2. **PostgreSQL (Required)**
   - Version: 13.0 or higher
   - Download: https://www.postgresql.org/download/windows/
   - Purpose: Database for storing trading data
   - Installation: Remember the superuser password

3. **Git (Optional)**
   - Version: Latest
   - Download: https://git-scm.com/download/win
   - Purpose: Version control and updates

### NPM Dependencies (Auto-installed)

The following packages will be automatically installed:

#### Core Framework
- express: Web server framework

#### Charts & Visualization

## Hardware Requirements

### Minimum Requirements
- **Operating System**: Windows 10 (64-bit) or Windows 11
- **Processor**: Intel i3 or AMD equivalent (2 cores)
- **Memory**: 4 GB RAM
- **Storage**: 1 GB free disk space
- **Network**: Internet connection for initial setup

### Recommended Requirements
- **Operating System**: Windows 11 (64-bit)
- **Processor**: Intel i5 or AMD equivalent (4+ cores)
- **Memory**: 8 GB RAM or more
- **Storage**: 2 GB free disk space (SSD recommended)
- **Network**: Stable internet connection

### Optimal Performance
- **Processor**: Intel i7/i9 or AMD Ryzen 7/9
- **Memory**: 16 GB RAM or more
- **Storage**: SSD with 5+ GB free space
- **Network**: High-speed internet

## Network Requirements

### Ports Used
- **5000**: Default application port (configurable)
- **5432**: PostgreSQL default port

### Firewall Configuration
- Allow Node.js through Windows Firewall
- Allow PostgreSQL through Windows Firewall
- No external network access required for basic operation

### Internet Requirements
- Required for initial setup and npm package installation
- Optional for Discord integration
- Optional for future market data integration

## Upgrade Path

### Updating Node.js
1. Download new version from nodejs.org
2. Run installer (will update existing installation)
3. Restart application

### Updating PostgreSQL
1. Backup database first
2. Download new version
3. Follow PostgreSQL upgrade documentation
4. Test application connectivity

### Updating Application
1. Backup current installation
2. Download new version
3. Copy .env file to new installation
4. Run setup.bat in new directory

Remember to always backup your data before making any system changes.
